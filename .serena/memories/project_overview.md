# Transport Daily Report - プロジェクト概要

## プロジェクトの目的
運送業向けの業務記録管理システム。GPS位置情報と連携した訪問記録、点呼記録、走行距離管理、PDF出力機能を提供する本格的なビジネスアプリケーション。

## 技術スタック
- **Flutter**: 3.29.3 (Stable Channel)
- **Dart**: ^3.7.2
- **プラットフォーム**: マルチプラットフォーム対応（Android, iOS, Web, Windows, macOS, Linux）
- **言語**: 日本語（完全ローカライズ済み）
- **業界**: 運送業・物流業

## 主要な機能（4つのタブ構成）
1. **📋 訪問記録管理**: GPS連携の顧客訪問記録
2. **🏢 得意先管理**: 顧客情報とGPS座標管理
3. **📊 点呼記録**: 法定要件に基づく始業・終業点呼記録
4. **📈 履歴・レポート**: PDF出力とデータエクスポート

## プロジェクト構造
```
lib/
├── models/          # データモデル（Client, VisitRecord, RollCallRecord, DailyRecord）
├── screens/         # UI画面（9画面）
├── services/        # ビジネスロジック（Storage, Location, PDF）
└── utils/          # プラットフォーム抽象化
```

## 重要な特徴
- **マルチプラットフォーム対応**: Web/ネイティブ分岐実装
- **オフライン対応**: ローカルストレージベース
- **位置情報サービス**: 高精度GPS取得
- **法的要件準拠**: 運送業法に基づく記録管理
