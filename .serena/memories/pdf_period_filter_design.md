# PDF期間絞り込み機能設計書

## 実装概要
既存のPDF出力機能に日別・月別・任意期間の絞り込み機能を追加

## 機能仕様

### 1. 期間選択タイプ
- **日別**: 特定の1日のデータを出力
- **月別**: 指定月の全データを出力  
- **任意期間**: 開始日〜終了日の範囲指定

### 2. UI実装
- `PeriodSelectorDialog`: 期間選択ダイアログ
  - TabBarで3つの選択モード切り替え
  - 日別: DatePicker
  - 月別: MonthPicker（年月選択）
  - 任意期間: DateRangePicker

### 3. データフィルタリング
- PDFサービスメソッドの拡張
- `filterRecordsByPeriod()`: 期間指定でのデータフィルタ関数
- 既存メソッドとの後方互換性保持

### 4. 対象画面
- `roll_call_list_screen.dart`: 点呼記録
- `visit_list_screen.dart`: 訪問記録

## 実装順序
1. 期間選択UIコンポーネント作成
2. データフィルタリング機能実装
3. 各画面への統合
4. テスト・動作確認
